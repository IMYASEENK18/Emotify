 
# 🎭 Emotify Flask Backend

Flask API for real-time emotion detection using MobileNetV2 and OpenCV.

---

## 📋 Features

- ✅ Face detection using OpenCV Haar Cascades
- ✅ Emotion recognition using MobileNetV2
- ✅ REST API for emotion detection
- ✅ Base64 image support
- ✅ CORS enabled for Express integration
- ✅ Modular architecture (easy to swap YOLO/better models later)
